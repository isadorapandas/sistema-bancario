# sistema-bancario
Projeto de um Sistema Bancário em POO com Python, realizado como um desafio durante as aulas de POO do bootcamp Luizalabs - Back-end com Python - 2º Edição, em abril de 2026.

O sistema possui um menu com as seguintes opções:

[d]     Depositar
[s]     Sacar
[e]     Ver extrato
[nc]    Criar nova conta
[lc]    Listar contas
[nu]    Criar novo usuário
[as]    Alterar senha da conta
[q]     Sair

Observações:
- Um cliente pode ter mais de uma conta, com senhas diferentes ou não.
- Quando uma nova conta é criada, é necessário informar uma senha para a mesma.
- A opção "Listar contas" mostra todas as contas no sistema.
- Nas operações de saque, extrato e alterar senha, é solicitada a senha da conta selecioanda. Caso a senha esteja incorreta, a operação não é realizada.
- O campo "nome da mãe" do cliente não possui preenchimento obrigatório.
